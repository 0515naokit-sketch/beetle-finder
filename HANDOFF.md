## 🛠 技術メモ

- Flask + Jinja2、Render.com で mainブランチへの push で自動デプロイ（2〜5分）
- **コード変更したら必ず git add → commit → push する**（CLAUDE.md のルール）
- sitemap.xml は app.py 内に定義。`D_TODAY` 変数で lastmod を一括管理
- 新ページ追加時は: ①テンプレート作成 ②app.pyにルート追加 ③app.pyのsitemapに登録 ④内部リンク追加
- 文字数カウントの確認方法:
  ```python
  python3 -c "import re; h=open('templates/X.html').read(); t=re.sub(r'<(script|style|head)[^>]*>.*?</\1>','',h,flags=re.S|re.I); print(len(re.sub(r'\s+',' ',re.sub(r'<[^>]+>',' ',t)).strip()))"
  ```
